MiniApps Privacy Policy

MiniApps does not intentionally collect personal information of any kind from anyone through this website or any apps developed by MiniApps.

Whether it is personally identifying information or aggregate anonymized information, MiniApps does not intentionally collect it.

Third party libraries that are used to develop any software will be listed in the description of the software it accompanies, and source code for any software developed by MiniApps will be made available on GitHub for public inspection.
